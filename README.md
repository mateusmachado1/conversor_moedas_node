Conversor de Moedas em Node.js
projeto simples de conversão de moedas no terminal, desenvolvido em JavaScript com Node.js.

O programa pergunta -> qual é a moeda base -> qual é a moeda de destino -> qual é o valor a ser convertido

Depois disso ele retorna o valor convertido

Moedas disponíveis

1 - Dólar (USD)

2 - Real (BRL)

3 - Euro (EUR)

Tecnologias usadas

JavaScript

Node.js

API de câmbio (ExchangeRate API)

projeto criado para praticar lógica de programação e JavaScript, simulando um conversor de moedas no terminal.
